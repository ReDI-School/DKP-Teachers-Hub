**📖 Was ist ein Computer? ·** **What is a Computer?**

A computer is a machine that can follow instructions and process information. It has four main parts:

• INPUT: things you use to give information TO the computer (keyboard, mouse, microphone)\
• OUTPUT: things the computer uses to show you results (screen, speakers, printer)\
• PROCESSING: the brain of the computer — the CPU (Central Processing Unit)\
• STORAGE: where the computer saves information (hard drive, USB stick, SSD)

Real-world comparison: A computer is like a chef. You give ingredients (input), the chef cooks (processing), and you get a meal (output)!

***

**📖 Cybersicherheit & Internetsicherheit ·** **Cybersecurity & Internet Safety**

The internet connects computers all around the world. But we need to stay safe online!

Important rules:\
• NEVER share your real name, address, school, or password with strangers online\
• Always tell a trusted adult if something online makes you feel uncomfortable\
• Not everything on the internet is true — always check with an adult\
• Use strong passwords and keep them secret

Being a good digital citizen means being safe, kind, and responsible online — just like in real life!

**📖 Was ist ein Algorithmus? · What is an Algorithm?**

An algorithm is a set of step-by-step instructions to solve a problem or complete a task. Computers follow algorithms exactly — they cannot guess or skip steps!

Example algorithm: How to make a sandwich

1. Get two slices of bread
2. Spread butter on one slice
3. Add your filling
4. Put the second slice on top
5. Done!

If you skip step 1, the computer (or sandwich maker!) would be confused. Order matters!

***

**📖 Block-Programmierung mit MakeCode · Block Programming with MakeCode**

MakeCode is a free coding platform where you drag and snap colourful blocks together to write programs — no typing needed!

The blocks are colour-coded by category:\
• BLUE blocks: Basic actions (show text, show number, show icon)\
• PURPLE blocks: Input (on button pressed, on shake)\
• GREEN blocks: Logic (if/else conditions)\
• RED blocks: Loops (repeat, forever)\
• ORANGE blocks: Variables (set, change)

Your program runs from top to bottom, just like reading a book!

Two important blocks:\
• 'on start' — runs ONCE when the program begins\
• 'forever' — keeps running in a loop forever

```
// Show a greeting when the program starts
on start:
  show string "Hallo!"
  pause 1000 ms
  show string "Ich bin ein Ninja!"
```

**📖 Was ist der Calliope Mini? · What is the Calliope Mini?**

The Calliope Mini is a small computer board — a real computer you can hold in your hand! It is the same as a big computer, just smaller and more colourful.

The Calliope Mini has:\
• 25 red LEDs to display numbers, letters, and symbols\
• 2 buttons (A and B) for input\
• Sensors for: movement (accelerometer), light, temperature, and compass direction\
• A speaker for sounds and music\
• A USB connector to connect to your laptop

You program it with MakeCode at: makecode.calliope.cc

***

**📖 Wie programmiere ich den Calliope Mini? · How to Program the Calliope Mini**

There are 3 steps to program your Calliope Mini:

1. PROGRAM: Write your program in MakeCode (makecode.calliope.cc)
2. SIMULATE: Test it in the simulator on screen first
3. DOWNLOAD: Click the download button and copy the file to your Calliope Mini

The Calliope Mini appears as a USB drive on your computer. Just drag the downloaded file onto it — like copying a file to a USB stick!

How to change the language in MakeCode: Click the gear icon (⚙️) in the top right corner.

```
// Blinkendes Herz (Blinking Heart)
// Put this inside a 'forever' block:
forever:
  show icon: heart
  pause 500 ms
  clear screen
  pause 500 ms
```

**📖 Was sind Sensoren? · What are Sensors?**

Sensors are things that allow computers or robots to feel their environment. But wait — humans have sensors too! Which ones do you have?

• Eyes = light sensor\
• Ears = sound sensor\
• Skin = temperature and touch sensor\
• Nose = smell sensor

The Calliope Mini has these sensors:\
• Compass sensor — measures direction (0°–360°)\
• Accelerometer — detects movement and tilt\
• Light sensor — measures brightness\
• Temperature sensor — measures temperature

Without sensors, the Calliope Mini could not 'notice' anything around it. Thanks to sensors, it can react — for example: when you face North, show an arrow ↑.

***

**📖 Der Kompass-Sensor & Wenn-dann-Bedingungen · The Compass Sensor & If/Else Conditions**

The Calliope Mini has a built-in compass sensor (it is inside the 'Lagesensor' chip). It measures direction from 0° to 360°:

• North = 0°\
• East = 90°\
• South = 180°\
• West = 270°

We use IF/ELSE conditions to make the Calliope Mini react:

IF (compass value < 180) → show 'N'\
ELSE → show 'S'

This is like a decision: if something is TRUE, do A. Otherwise, do B.

```
// Calliope as a Compass
forever:
  if compass heading < 180:
    show string "N"
  else:
    show string "S"
```

• IF the compass value is less than 90 OR greater than 270 → show an arrow ↑ (you are facing North!)

**💻 Calliope als Kompass · Calliope as a Compass**

**📖 Was ist eine Schleife? · What is a Loop?**

A loop is a way to repeat instructions multiple times without writing them out again and again.

Loops in real life:\
• Your heartbeat repeats — that is a loop!\
• Your weekly timetable repeats every week — that is a loop!\
• Breathing in and out — that is a loop!\
• Walking: step, step, step — that is a loop!

Without loops, if you wanted to blink an LED 50 times, you would need to write 100 blocks! With a loop, you only need 3 blocks.

What happens if we give a loop no stop rule? It runs forever!

***

**📖 Arten von Schleifen in MakeCode · Types of Loops in MakeCode**

MakeCode has several loop blocks (all in the red Loops category):

• FOREVER (Dauerhaft): Runs forever until the device is turned off\
• REPEAT X TIMES (Wiederhole X-mal): Runs a specific number of times then stops\
• FOR INDEX FROM 0 TO X: Counts up from 0 and runs each time

The most common ones for beginners are FOREVER and REPEAT.

```
// Übung 1: Lichtshow mit Schleifen
// Use 'repeat 4 times' block:
repeat 4 times:
  show icon: heart
  pause 300 ms
  show icon: small heart
  pause 300 ms

// Übung 2: Zählen mit Loops
// Use 'for index from 0 to 4':
on start:
  for index from 0 to 4:
    show number: index
    pause 500 ms
// This counts: 0, 1, 2, 3, 4!
```

**📖 Wie Computer Daten speichern · How Computers Store Data**

Computers have two types of memory:

• RAM (Arbeitsspeicher / Working Memory): Where the computer stores things it is currently working on. Like a desk — you put things on it while you work, but when you turn off the computer, it is cleared!

• SSD / Hard Drive (Festplatte / Permanent Storage): Where the computer saves things permanently. Like a shelf — things stay there even when you turn off the computer.

When you write a program in MakeCode, variables are stored in RAM — they exist while the program runs.

***

**📖 Was ist eine Variable? · What is a Variable?**

A variable is like a labelled box that stores information. You can put something in the box, look at what is inside, and change what is inside at any time.

Real-life examples:\
• A scoreboard at a football game — it stores the score and changes when a goal is scored\
• Your age — it is stored as a number and changes every birthday\
• A shopping list — it stores items and you can add or remove things

In MakeCode, variables are in the orange Variables category:\
• 'set score to 0' — give the variable a starting value\
• 'change score by 1' — add 1 to the variable\
• 'show number score' — display the current value

```
// Score counter: A adds points, B resets
on start:
  set score to 0
  show number score

on button A pressed:
  change score by 1
  show number score

on button B pressed:
  set score to 0
  show number score
```
